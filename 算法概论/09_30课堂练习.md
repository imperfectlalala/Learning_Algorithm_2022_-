## 3.找出一个满足下列竖式的五位数，输出该五位数及相应六位数结果
```PYTHON
def find_six():
  #乘积是6位数，A得大于2
  for A in range(3,10):
      A = int(A)
      for D in range(1,10):
          D = int(D)
          E = D*100000+D*10000+D*1000+D*100+D*10+D
          if E%A==0:
              F = E/A
              if F/10000==A and (F%100)/10==A:
                  if (F/1000)%10==F%10:
                      print("{}*{}={}".format(F,A,E))
```
